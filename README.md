# JB Solutions — Marketing site

This repository contains a static React + Vite marketing site for JB Solutions. It's designed for deployment to GitHub Pages.

Getting started:

- Install dependencies: npm install
- Run dev server: npm run dev
- Build for production: npm run build

Deploying to GitHub Pages:

1. Build the site: npm run build
2. Copy the contents of dist/ to your gh-pages branch or configure GitHub Pages to serve from the repository's Pages settings (e.g., from the gh-pages branch or the docs/ folder). Because this is a static site, you can also push the built files to the gh-pages branch.

Notes:
- The contact CTAs use a mailto: link to jbsolutionslabs@gmail.com
- Add a favicon at public/favicon-32.png before deploying.
